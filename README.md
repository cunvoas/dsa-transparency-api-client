# DSA/DMA Transparency API Client

This project is an API client designed for compliance with the Digital Services Act (DSA) and Digital Markets Act (DMA) regulations.

## Features

- **Transparent API Calls**: The client provides seamless and transparent access to the relevant APIs required by DSA/DMA.
- **Legal Bundle Export**: Generates the required legal bundle (liasse légale) in XLSX format for regulatory reporting and compliance.

## Usage

1. Integrate the client into your application or workflow.
2. Use the provided methods to interact with the DSA/DMA APIs.
3. Export the legal bundle as an XLSX file when needed for compliance or audits.
